## 二、定位策略

### 2.1 核心定位

**"Прямые поставки пуэра из Юньнани — без посредников"**
（云南普洱茶直供——无中间商）

### 2.2 差异化卖点

| 维度 | 竞品现状 | PUERH DIRECT 优势 |
|------|---------|-------------------|
| 供应链 | 芳村/莫斯科中间商 | 云南产地直发 |
| 价格 | 2-3 层加价 | 出厂价，B2B 批发价 |
| 品类 | 泛中国茶 | 专注普洱茶，44 SKU |
| 品质 | 无法溯源 | 产地 + 树龄 + 年份透明 |
| 量级 | 零售散卖 | 批发级 MOQ，承接大批量 |
| 定制 | 无 | 白标/贴牌 OEM 服务 |

### 2.3 目标客户画像

| 客户类型 | 决策人 | 需求 | 触达方式 |
|---------|--------|------|---------|
| 茶叶零售店 | 店主/采购 | 稳定品质、合理批发价 | VK 行业群组+私信 |
| 餐厅/酒吧 | 主厨/老板 | 茶单定制、稳定供应 | 餐饮群组渗透 |
| 礼品公司 | 采购经理 | 批量定制、包装方案 | 企业礼品群组 |
| 茶空间/茶馆 | 创始人 | 特色茶品、产地故事 | 开业群组互动 |
| 批发分销商 | CEO/采购总监 | 大批量、低价格 | VK 广告+直接私信 |

**渗透四步法：**
1. 加入群组，前 3 天静默观察群规和活跃成员
2. 第 4-7 天开始在相关帖子下做专业评论（不推销）
3. 第 2 周开始发布有价值内容（行业洞察、产品知识）
4. 当有人提问茶叶相关问题时，专业解答 → 引导私信

**广告目标：** 获取 B2B 询盘（Lead Form / 私信）

- 地域：莫斯科、圣彼得堡、新西伯利亚、叶卡捷琳堡（优先）

- 测试期：500-1000 卢布/天（约 40-80 元）

## 六、注意事项

1. **EAC 认证**：提前准备，这是对接正规 B2B 客户的必要门槛
2. **俄语本地化**：所有内容（帖子、私信、目录）必须俄语优先，英文辅助
3. **响应速度**：俄罗斯客户期望 2 小时内回复，延迟 = 丢单
4. **账号安全**：新号前两周不要大量加好友/发私信，每天 ≤ 15 条，逐步增加
5. **广告合规**：VK 广告必须标注 "Реклама"，避免使用 VPN 登录
6. **竞争壁垒**：快速建立"产地直供"品牌认知，因为目前 VK 上几乎没有直接竞争者
