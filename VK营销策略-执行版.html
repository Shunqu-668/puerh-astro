级别定义响应处理

❌ 无效广告号/竞品—忽略

标准转化路径

VK 帖子/私信→ 加 Telegram→ 发俄语目录 PDF→ 确认需求→ 报价→ 寄样品→ 付款→ 发货

跟进来电节奏

  • A 级 → 当天 Telegram → 24h 报价 → 3 天 follow-up
  • B 级 → 当天私信 → 3 天 follow-up → 7 天再推一条
  • C 级 → 加入"潜在客户"列表 → 新品/降价时群发
